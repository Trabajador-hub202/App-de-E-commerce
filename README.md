# 🦁 La Casa del León 
-App Mobile E-Commerce & WhatsApp Ordering System
Aplicación móvil híbrida desarrollada en respuesta a un brief de negocio estratégico cuyo objetivo es incrementar la clientela y automatizar la recepción de pedidos en tiempo real para un restaurante, eliminando costos de infraestructura de servidores mediante integraciones externas.

🛠️ Stack Tecnológico & Arquitectura*¿   
*Framework: Flutter (Dart)
*   Integraciones:WhatsApp Business API / Deep Linking (url_launcher)
*   Manejo de Estado: Gestión de Estado Reactivo para el flujo del carrito.
*   *Diseño: UI adaptada con assets de alta definición y Material Design Cards.
⚙️ Funcionalidades Clave*
Catálogo Digital Interactivo: Presentación de menú optimizado en tarjetas visuales con control de inventario y precios dinámicos.
Carrito de Compras Reactivo: Flujo optimizado de selección de productos y cálculo automático de totales sin persistencia pesada (máximo rendimiento en memoria).
Validación Estricta de Formularios: Módulo de checkout con validación en tiempo real de campos críticos (Nombre, Teléfono, Dirección) que bloquea el flujo si la información está incompleta.
*Integración de API de WhatsApp: Generación de enlaces dinámicos con formato de texto limpio (URL Encoding) para transferir los datos del pedido directamente al chat del comercio.
